# CodeWorkApp

## Installation

`npm install`

`npx expo start`

We display job postings thanks to the data we pull from the API. We can go to the detail page of these job postings and see the details of the job posting we want, and if we want, we can add it to our favorite job postings or go to the job posting site. On our favorite job posting page, we can see all our favorite job postings and we can delete those job postings if we wish. We keep your favorite jobs in global state (redux).

Apiden çektiğimiz veriler sayesinde iş ilanlarını gösteriyoruz. Bu iş ilanlarının detay sayfasına gidip istediğimiz iş ilanının detaylarını görebiliyoruz ve istersek favori iş ilanlarımıza ekleyebiliyoruz veya iş ilan sitesine gidebiliyoruz. Favori ilan sayfamızda favorilediğimiz tüm iş ilanları görebiliriz ve istersek o iş ilanlarını silebiliriz. Favori iş ilanlarını global state'de (redux) tutuyoruz.
